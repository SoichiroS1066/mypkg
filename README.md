# mypkg
![test](https://github.com/ryuichiueda/robosys2022/actions/workflows/test.yml/badge.svg)

## 機能
* talker.pyで1ずつ加算されるnをlistener.pyで表示する

## 起動の手順
`$ cd ~/ros2_ws/src
$ git clone https://github.com/SoichiroS1066/mypkg.git
$ cd ~/ros2_ws/
$ colcon build`

## 使用例
$ ros2 run mypkg listener



## 必要なソフトウェア
* Python 
  * テスト済み: 3.7 ～ 3.10

## テスト環境

* Python 3.7 ～ 3.10
